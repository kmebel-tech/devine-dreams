## Deployment Instructions for GoDaddy

### Method 1: Using GoDaddy File Manager (Recommended for beginners)

1. **Log in to your GoDaddy account**
   - Go to https://www.godaddy.com
   - Sign in with your credentials

2. **Access your hosting account**
   - Navigate to "My Products"
   - Find your Web Hosting account
   - Click "Manage" next to your hosting plan

3. **Open File Manager**
   - In the hosting control panel (cPanel), find and click "File Manager"
   - Navigate to the `public_html` folder (or `www` folder, depending on your setup)

4. **Upload your files**
   - Click the "Upload" button
   - Select and upload both `index.html` and `styles.css`
   - Make sure they are in the root of `public_html` (not in a subfolder)

5. **Verify the upload**
   - Once uploaded, you should see both files listed in the File Manager
   - The file permissions should be set to 644 (default)

6. **Test your website**
   - Open a web browser and go to www.devinedreams.com
   - Your website should now be live

### Method 2: Using FTP (For advanced users)

1. **Get your FTP credentials**
   - In your GoDaddy hosting control panel, find FTP credentials
   - Note your FTP hostname, username, and password

2. **Connect using an FTP client**
   - Download FileZilla or another FTP client if needed
   - Connect to your server using the credentials

3. **Upload files**
   - Navigate to the `public_html` folder on the remote server
   - Upload `index.html` and `styles.css` to this folder

4. **Test your website**
   - Visit www.devinedreams.com in your browser

## Customization

To update the website content:

1. Open `index.html` in any text editor
2. Find the section you want to modify
3. Update the text content between the HTML tags
4. Save the file and re-upload to GoDaddy

To change colors or styling:

1. Open `styles.css` in any text editor
2. Modify the CSS variables at the top of the file under `:root`
3. Save and re-upload to GoDaddy
